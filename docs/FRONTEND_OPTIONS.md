# Куда выводить интерфейс (без своего домена)

Сейчас embed на сайт заказчика **не делаем**. Варианты:

## 1. Vercel + этот репозиторий (рекомендуется)

- Уже настроено: `vercel.json`, Vite  
- Бесплатный URL: `https://….vercel.app`  
- Чат: `/` или `/c/customer-demo`  
- Кабинет: `/login` → `/app`  
- **Плюсы:** один код, git push → деплой, рядом с Supabase  
- **Минусы:** нужен аккаунт Vercel  

## 2. Bolt (Replit / bolt.new)

- Можно импортировать UI для быстрого прототипа чата  
- **Плюсы:** быстро «нарисовать»  
- **Минусы:** второй контур, сложнее синхронизировать auth + админку + Supabase; не рекомендуем как основу Office  

## 3. Tilda / сайт заказчика — позже

- Embed iframe или кнопка «Открыть чат» на ваш Vercel URL  
- Свой домен — когда появится: CNAME на Vercel  

## 4. Локально

```powershell
npm run dev
```

Для показа заказчику — достаточно Vercel preview URL.
